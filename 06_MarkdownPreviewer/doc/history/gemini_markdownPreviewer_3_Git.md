ターミナルで以下のコマンドを実行してください。

# 3. コミットを作成

text
remote: Create a pull request for '06AppCreate' on GitHub by visiting:

Gitのコミットログをグラフ形式で表示するには、git log コマンドに --graph オプションを付けて使用します。

より見やすくするために、以下のオプションを組み合わせて使うのが一般的です。

推奨コマンド
bash
git log --graph --oneline --all --decorate
オプションの解説
--graph: 左端にコミット履歴の分岐やマージをアスキーアートの線で表示します。
--oneline: 各コミットを1行に短縮して表示します（ハッシュとタイトルのみ）。
--all: 現在のブランチだけでなく、すべてのブランチの履歴を表示します。
--decorate: ブランチ名やタグ名をコミットの横に表示します。
さらに詳細なグラフを表示したい場合
日時や作者名も含めてグラフ表示したい場合は、以下のようなフォーマット指定を使うこともできます。

bash
git log --graph --pretty=format:'%h -%d %s (%cr) <%an>' --abbrev-commit
